PART III — Reflection 

Answer the following questions in your own words: 

1. What is your website about, and why did you choose this topic? 

My website is a personal portfolio highlighting my interest in becoming a cardiologist along with my personal advocacy of the phrase “prevention is better than cure”. I chose this topic because building this website felt like I am one step towards achieving my ambitions. Creating this project motivated me to look forward to what more I can do in my future career.

2. What was the most difficult part of building your site, and how did you get through it? 

The most difficult part of creating this website was building the CSS style. For the website to work efficiently, it requires a lot of adjustments with both the HTML and CSS style layout. This part can become confusing with all the codes and designs. I overcome this by slowly learning the specific use of various codes.

3. What is one thing you are proud of in your finished website? 

Creating this website taught me how to work through complex codes.I am proud of learning a new thing I could possibly apply and explore more in future projects. This project was full of learnings which encourages me to explore more into different skills I haven’t tried.

4. If you had more time, what would you improve or add? 

If I was given more time to create this project I would expand this website by creating better animations and adding more interactive layouts. I look forward to improving my CSS coding and creating a cleaner coding process.
